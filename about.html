About me!

Hi! My name is Franek Borkowski and I'm a 15 year old developer from Poland. I started my journey with coding in primary school making small HTML projects in 2020 for homework. After some time I've got more interested in coding and because of that I looked more into web design and learning languages as python or C++. Then, after my exams I went to a high-school called ZSTI in Gliwice on a programming major. Since then I can code in HTML, CSS, PHP, JavaScript, SQL, C++ and Python on a basic level. I'm mainly concentrated on web design and making the best websites I possibly can. I hope that in the future I can work for small and big companies and make the best of them. I don't really know what exactly I will do, whether it will be front-end or back-end, but I'm sure that programming is something I'm passionate about and I know I want to dedicate my life to it.
